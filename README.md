# Mom - Mother Repository

This repository contains the complete transferred content from the [miketui/Fm](https://github.com/miketui/Fm) repository, including all rebranded output files and OEBPS folders.

## Contents

### OEBPS Folder (208 files)
The OEBPS (Open eBook Publication Structure) folder contains the complete structure for "The Artisan's Path" ebook:

- **content.opf** - The package document that describes the publication
- **ebook/** - Complete ebook structure with metadata
- **fonts/** - Custom web fonts (Cinzel Decorative, Montserrat, Libre Baskerville)
- **images/** - All visual assets including chapter quotes, decorative elements, and author photo
- **styles/** - CSS stylesheets (fonts.css, print.css, style.css)
- **text/** - All XHTML content files including:
  - Title page, copyright, table of contents
  - 16 chapters organized into 4 parts
  - Preface, conclusion, acknowledgments
  - Interactive elements (quizzes, journaling pages, assessments)
  - Navigation document

### REBRANDED_OUTPUT Folder (309 files)
The main rebranded output folder containing:

- **The-Artisans-Path.epub** - Complete compiled EPUB file (84.8 MB)
- **content.opf** - Package document
- **mimetype** - EPUB mimetype declaration
- **META-INF/** - Container metadata
- **fonts/** - Web fonts for the ebook
- **images/** - All image assets
- **styles/** - CSS stylesheets
- **xhtml/** - XHTML content files
- **templates/** - Chapter templates
- **scripts/** - Build and automation scripts
- **pdf-pod/** - Print-on-demand PDF files
- **react-components/** - React-based components
- **dist/** - Distribution files
- **.claude/** - Claude AI project files
- Documentation files (README.md, automation guides, completion reports)

### REBRANDED-output Folder (101 files)
Alternative rebranded output structure with:

- **fonts/** - Font files
- **images/** - Image assets
- **styles/** - Stylesheets
- **xhtml/** - XHTML content files

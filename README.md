# Mom - Mother Repository

This repository contains the transferred OEBPS folder from the [miketui/Fm](https://github.com/miketui/Fm) repository.

## Contents

### OEBPS Folder
The OEBPS (Open eBook Publication Structure) folder contains the complete structure for "The Artisan's Path" ebook, including:

- **content.opf** - The package document that describes the publication
- **ebook/** - Complete ebook structure with metadata
- **fonts/** - Custom web fonts (Cinzel Decorative, Montserrat, Libre Baskerville)
- **images/** - All visual assets including chapter quotes, decorative elements, and author photo
- **styles/** - CSS stylesheets (fonts.css, print.css, style.css)
- **text/** - All XHTML content files including:
  - Title page, copyright, table of contents
  - 16 chapters organized into 4 parts
  - Preface, conclusion, acknowledgments
  - Interactive elements (quizzes, journaling pages, assessments)
  - Navigation document

## Structure

The content is organized into four main parts:
- Part I: Foundations of Creative Hairstyling
- Part II: Building Your Professional Practice
- Part III: Advanced Business Strategies
- Part IV: Future-Focused Growth

This content was transferred from the rebranded output of the Fm repository, preserving the complete OEBPS structure for the ebook.
